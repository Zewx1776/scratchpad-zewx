# Auradin Build - Quick Start Guide

## Overview
The Auradin (Holy Light Aura) build focuses on dealing holy damage through auras and AOE effects. This build excels at clearing groups of enemies with sustained holy damage.

**Build Reference:** [Mobalytics Auradin Holy Light Aura Build](https://mobalytics.gg/diablo-4/builds/auradin-holy-light-aura-paladin)

## Quick Setup (5 Minutes)

### Step 1: Enable Core Abilities
1. Open the in-game menu
2. Navigate to **Paladin: Winterz|Karnage V1**
3. Enable the plugin with the main toggle
4. Go to **Active Skills**
5. Enable these abilities:
   - ✅ **Holy Light Aura** - Your main AOE damage source
   - ✅ **Basic Strike** - Filler attack
   - ✅ **Fist of the Heavens** - Secondary AOE (optional)

### Step 2: Configure Holy Light Aura
1. Open **Holy Light Aura** settings
2. Set **Spell Range**: 8.0 (adjust based on your comfort)
3. Set **Min Enemies**: 3 (cast when 3+ enemies nearby)

### Step 3: Configure Basic Strike
1. Open **Basic Strike** settings
2. Set **Spell Range**: 3.0
3. Keep it enabled as your filler

### Step 4: Set Targeting Mode
1. In the main menu, find **Targeting Mode**
2. Choose **"player"** for easier automated play
3. Or choose **"cursor"** for manual control

### Step 5: Test It Out!
1. Enter combat
2. Activate **orbwalker** mode (combo/clear)
3. Watch the plugin cast your abilities automatically
4. Check console for spell feedback

## Rotation Priority

**Auradin Build Rotation:**
```
Holy Light Aura (when 3+ enemies) → Fist of Heavens (if enabled) → Basic Strike (filler)
```

## Recommended Settings

### Holy Light Aura
- **Enable**: ✅ Yes
- **Spell Range**: 8.0
- **Min Enemies**: 3

### Basic Strike
- **Enable**: ✅ Yes
- **Spell Range**: 3.0

### Optional Skills
- **Fist of the Heavens**: Enable for extra burst AOE
  - Min Enemies: 3-5

## Tips for Auradin Build

1. **Positioning**: Stay in the middle of enemy groups to maximize aura damage
2. **Min Enemies**: Lower the value to 2-3 for faster clearing
3. **Range**: Keep Holy Light Aura range at 8.0 for good coverage
4. **Targeting**: Player mode works best for this build
5. **Movement**: Use auto-play for automatic positioning

## Troubleshooting

**Holy Light Aura not casting:**
- Check min enemies setting (lower it to 2)
- Verify spell is enabled
- Make sure orbwalker is active

**Not enough damage:**
- Enable additional AOE spells (Fist of Heavens)
- Ensure your character has proper gear
- Check that spell IDs are correct for your game

**Spell IDs:**
- Remember to update spell IDs in the .lua files with actual game IDs
- Current IDs are placeholders

## Build Tips

- Focus on holy damage gear
- Look for AOE damage increases
- Stack area effect radius
- Prioritize cooldown reduction
- Get movement speed for repositioning
