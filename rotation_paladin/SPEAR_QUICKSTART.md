# Spear of the Heavens Build - Quick Start Guide

## Overview
The Spear of the Heavens build focuses on calling down divine spears from above for devastating ranged AOE damage. This build excels at long-range combat and kiting.

**Build Reference:** [Mobalytics Spear of the Heavens Build](https://mobalytics.gg/diablo-4/builds/spear-of-the-heavens-paladin)

## Quick Setup (5 Minutes)

### Step 1: Enable Core Abilities
1. Open the in-game menu
2. Navigate to **Paladin: Winterz|Karnage V1**
3. Enable the plugin with the main toggle
4. Go to **Active Skills**
5. Enable these abilities:
   - ✅ **Spear of the Heavens** - Your main damage dealer
   - ✅ **Basic Strike** - Filler attack
   - ✅ **Judgement** - Secondary AOE (optional)

### Step 2: Configure Spear of the Heavens
1. Open **Spear of the Heavens** settings
2. Set **Spell Range**: 12.0 (long range)
3. Set **Min Enemies**: 2 (cast when 2+ enemies)

### Step 3: Configure Basic Strike
1. Open **Basic Strike** settings
2. Set **Spell Range**: 3.0
3. Keep it enabled as your filler

### Step 4: Set Targeting Mode
1. In the main menu, find **Targeting Mode**
2. Choose **"cursor"** for precise spear placement
3. Or choose **"player"** for automated targeting

### Step 5: Test It Out!
1. Enter combat
2. Activate **orbwalker** mode (combo/clear)
3. Watch divine spears rain down on enemies
4. Check console for spell feedback

## Rotation Priority

**Spear Build Rotation:**
```
Spear of the Heavens (when 2+ enemies) → Judgement (if enabled) → Basic Strike (filler)
```

## Recommended Settings

### Spear of the Heavens
- **Enable**: ✅ Yes
- **Spell Range**: 12.0
- **Min Enemies**: 2

### Basic Strike
- **Enable**: ✅ Yes
- **Spell Range**: 3.0

### Optional Skills
- **Judgement**: Enable for additional AOE control
  - Spell Range: 10.0
  - Min Enemies: 2-3

## Tips for Spear Build

1. **Range**: Take advantage of 12.0 range to stay safe
2. **Cursor Mode**: Use cursor targeting for precise AOE placement
3. **Kiting**: This build is excellent for hit-and-run tactics
4. **Min Enemies**: Keep at 2 for consistent casts
5. **Positioning**: Stay at max range for safety

## Troubleshooting

**Spears not casting:**
- Check spell range (ensure enemies are within 12.0)
- Verify min enemies is set to 2
- Make sure targeting mode is appropriate

**Not hitting enemies:**
- Use cursor mode for better accuracy
- Enemies might be moving - use prediction
- Reduce spell range if needed

**Spell IDs:**
- Update spell IDs in spear_of_the_heavens.lua
- Current IDs are placeholders

## Build Tips

- Focus on ranged damage bonuses
- Look for AOE damage increases
- Stack cooldown reduction
- Get cast speed for faster spears
- Prioritize survivability for kiting

## Advanced Tips

2. **Burst Combo**: Add Fist of Heavens for burst windows
3. **Mobility**: Keep basic strike for close-range emergencies
4. **Auto-Play**: Works great with auto-play for farming
